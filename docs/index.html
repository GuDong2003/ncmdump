What changed

Static deployment, native core logic.

  • The page reuses the NCM parsing and key-box flow from the C++ implementation.
  • Decoded audio is generated in your browser and offered as a direct download.
  • Cover art and metadata are extracted when present in the source file.
  • Browser mode does not rewrite ID3 or FLAC tags like the native CLI does.
